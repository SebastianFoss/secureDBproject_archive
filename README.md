# secureDBproject_archive

Seattle University Fall quarter 2024-2025 CPSC 3300

Fundamentals of Databases Final Project:

Secure Credit Card Database

Features:
- Securely store customer credit card information!
- Users can log in to see their information
- Root user can log in and run database queries, pre-written and ad-hoc queries supported

Security:
- SHA-256 hashed user passwords
- input validation
- 2NF normalization

languages: JS, php, html/css, SQL
tools: MySQL workbench, WinSCP (SFTP), VScode

DB server: CS1 at Seattle University
